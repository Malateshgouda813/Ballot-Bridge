# Ballot-Bridge


A secure and user-friendly "Ballot-Bridge" developed using "PHP, MySQL, HTML, CSS, and JavaScript". 

The system allows users to register, log in, and vote for candidates digitally while ensuring that each voter can vote only once.

The "Ballot-Bridge" is deployed online using InfinityFree hosting and is accessible via a live web environment.

# Features

• User Registration with profile photo  
• Secure Login System  
• One-person-one-vote functionality  
• Live Voting Results Dashboard  
• Graphical vote visualization  
• Current leading candidate display  
• Download voting results  
• Separate roles for "Voters" and "Candidates" 
• MySQL database integration  



# System Screens

* Registration Page
Users can create an account by entering personal details and uploading a profile image.
* Dashboard
Displays voter profile, voting status, and candidates list.
* Voting Section
Users can vote for their preferred candidate.
* Results Section
Displays vote counts and the leading candidate with chart visualization.

# 🔐 Security Features

• Prevents multiple voting by the same user  
• Role-based access control  
• Session authentication  
• Environment variables using `.env`  
• Secure database connection
